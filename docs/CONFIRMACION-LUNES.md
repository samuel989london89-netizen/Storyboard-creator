# 2894_signals — Confirmacion cada lunes (sin gasto automatico)

## Como funciona ahora

**Nada se genera ni se envia solo.** Tu controlas todo.

### Cada lunes (gratis)

GitHub te crea un **recordatorio** (issue) y puede llegarte un email de GitHub.
No gasta creditos de Cursor.

### Tu confirmas en Cursor

Abre el chat de Cursor y escribe:

```
Aprobado — genera y envia 2894_signals de esta semana
```

O simplemente:

```
Aprobado
```

### Que pasa despues de tu Aprobado

1. Se genera el informe (usa creditos Cursor — solo cuando tu apruebas)
2. Se marca aprobacion en `config/approval.json`
3. GitHub envia el email a:
   - Slondon99@outlook.es
   - juliosatorre@gmail.com
4. La aprobacion se resetea (no se puede reenviar sin nuevo Aprobado)

---

## Si no quieres enviar esta semana

No escribas nada. No pasa nada. Cero coste.

---

## Comandos utiles

| Quieres | Escribe en Cursor |
|---------|-------------------|
| Producir y enviar | `Aprobado` |
| Solo ver informe sin enviar | `Genera el informe pero no lo envies` |
| Cancelar | `Cancelar newsletter esta semana` |

## Recordatorio

Cursor no puede enviarte un mensaje dentro del chat solo los lunes.
El aviso llega por **GitHub** (issue + notificacion email de GitHub si las tienes activas).
