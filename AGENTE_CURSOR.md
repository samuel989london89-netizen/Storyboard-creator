# Tu agente de noticias IA creativa en Cursor (simple)

No necesitas saber que es un repo para usar esto.

## Que te dejo ya preparado

1) Prompt del agente:
- `prompts/ai-news-creative.md`

2) Este manual rapido:
- `AGENTE_CURSOR.md`

## Como usarlo (solo copiar y pegar en el chat)

Cada semana escribe exactamente:

`Lee prompts/ai-news-creative.md, investiga las novedades de esta semana y genera el informe. Luego guardalo en reports/YYYY-MM-DD-ai-news.md`

Ejemplo real:

`Lee prompts/ai-news-creative.md, investiga las novedades de esta semana y genera el informe. Luego guardalo en reports/2026-06-17-ai-news.md`

## Version corta para compartir por WhatsApp/Slack

Despues del informe, pide:

`Ahora dame una version de 10 lineas para enviar al equipo creativo.`

## Newsletter automatica (correo semanal)

Guia completa: `docs/SETUP-NEWSLETTER.md`

Resumen:
- 1 informe/semana para max 5 afiliados
- Envio por email gratis (Resend + GitHub Actions)
- Proteccion de creditos Cursor con `config/budget.json`
